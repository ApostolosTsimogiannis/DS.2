# DS.2 RC Helicopter
DS.2 is a mostly 3D printable 570 size RC helicopter.
All files are downloadable for users to build this heli. Read the manual section for a complete build instruction.

![manual](https://github.com/user-attachments/assets/beab5a99-be0c-4505-8aad-56fd445ac59c)

# Technical Specifications:

|Description|Value|
|---|---|
| Length | 1120mm |
| Height | 355mm |
|Width (skids)|  190mm|
|Weight (no battery)| 2180kg (1)|
|Motor Size| 4020-4025|
|Servos| Both 15mm and 20mm servos can be used|
|Main blade Size| 560-575mm (2)|
|Tail blade size| 85-95mm (2)|
|Battery|6s 4000-5600mAh|

(1) With 15mm servos

(2) Depending on tail belt length.
